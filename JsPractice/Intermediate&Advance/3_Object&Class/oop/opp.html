Constructor Function

A constructor function in JavaScript is a special type of function used to create and initialize objects with the same properties and methods. Constructor functions are typically used to create multiple objects with a similar structure. By convention, the name of a constructor function starts with an uppercase letter to differentiate it from regular functions.

How Constructor Functions Work

Define: Constructor functions are defined like regular functions but are intended to be used with the new keyword.
